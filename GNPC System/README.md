# GNPC Digital Operations Portal

A web-based Digital Operations Management System developed for the **Ghana National Petroleum Corporation (GNPC)** as an academic prototype.  
The system supports secure login, real-time operational monitoring, dashboard analytics, and petroleum sector reporting.
